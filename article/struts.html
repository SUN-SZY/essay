Ⅰ. 上课笔记

一、值栈特点

  • 符合栈的基本属性

    • 先进后出，后进先出。
    • 顶部元素称为栈顶，底部元素称为栈底

  • debug调试方式

  • Action的对象类只有一个Context属于map集合，里面为Action对象的引用

  • 页面中加入<s:debug></s:debug>

  • root — 调试出的页面 List、map等集合，栈顶的元素为Action对象的引用，而不是对象

  • Context ---- 存储的Key和value的值

二、值栈的储存

  • set方法

  • 获取ActionContext类对象

    • ACtionContext context=ActionContext。getContext（）； Values

  • 第三种方法

    • 在Action中生成get方法
      • 定义变量名称
      • 生成get方法
      • 在默认方法中加入定义
      • 在Struts文件中配置
    • 存放对象
      • 定义一个对象
      • 生成对象变量的get方法
      • 默认方法中进行设置
      • user.setUsername("")设定对应的值
    • 集合
      • 定义list集合变量
      • privat List<User> list = new ArrayList<User>()
      • 返回get方法
      • 默认方法中设置User的值
      • 第一组数据 User user=new User(); User.setUsername("") 第二组数据 User user2=new User（）: setUsername("") 向数组中加入数据 list.add("user") list.add("user2")

  • jsp页面的取值方法

    • 第一种 获取list值栈中的数据
      • <s:property value ="list[0].username"></s:property>
    • 第二种 遍历list得到list里面的每个user对象
      • <s:iterator value="list"> <s:property value="username"></s:property>
    • 第三种 遍历值栈list集合，得到每个user对象
      • 执行方式： 把每次遍历出来的user对象放到context里面获取context里面的数据特点，写ognl表达式使用特殊符号#
      • <s;property value="#user.username"></s:property>

  • 获取数据方式

    • 定义对象，默认方法赋值之后如何获取数据
      • 获取对象里面的数据
      • <s:property value="user.username"></s:property>
    • 定义变量，获取变量中的数据
      • 获取字符串的值
      • <s: peroperty value="name">
    • 获取strack.set中的数据
      • 获取值中的数据
      • <s:property value"username"></s:porperty>
    • 获取push中的值
      • 获取push值中的数据
      • <s:property value="[0].top"></s:porperty>

Ⅱ. 实验记录

  • 值栈的三种输入方式

  • 第一种

     //使用值栈对象里面的set方法
   	  //1.获取值栈对象
        ActionContext 		  
        context=ActionContext.getContext();
        ValueStack stack=context.getValueStack();
        //利用set方法向值栈中放数据
        stack.set("username","SUN");
        //利用push方法向值栈中放数据
       stack.push("abcdefg");

    • 输入方式

      • 通过set方法向值栈中存入数据

    • 输出方式

      • 字符串类型：<s:property value="username"></s:property>
      • push类型：<s:property value="[0].top"></s:property>

  • 第二种

    //定义变量
private  String name;
//生成get方法
  public String getName() {
     return name;
 }
 public String execute()  {
      name="SUN";
     return "success";

  }

  • 输入方式 - 通过定义变量使用get方法存入数据 - 输出方式

    • 输出栈中字符串：<s:property value="name"></s:property>

  • 第三种

private List<User> list=new ArrayList<User>();
           public List<User> getList() {
               return list;
           }
         
           public String execute()  {
         
               User user =new User();
               user.setUsername("SUN");
               user.setPassword("113256");
               user.setAddress("earth");
         
               User user1 =new User();
               user1.setUsername("ZHAN");
               user1.setPassword("123456");
               user1.setAddress("earth");
         
               User user2 =new User();
               user2.setUsername("YANG");
               user2.setPassword("11236");
               user2.setAddress("earth");
         
               list.add(user);
               list.add(user1);
               list.add(user2);
         
               return "success";
         
           }

  • 输入方式

    • 利用list数组方式方式进行输入

  • 输出方式

    • 陈列输出
